# Endurance Portfolio

A modern, responsive portfolio website for Endurance Banigo, showcasing her skills as a Tech Creative and Digital Strategist.

## Features

- **Modern Design**: Clean, professional layout with gradient backgrounds and smooth animations
- **Responsive**: Optimized for desktop, tablet, and mobile devices
- **Interactive Elements**: Smooth scrolling navigation, hover effects, and reveal animations
- **Color Scheme**: Professional blue and gray palette with accent colors
- **Sections**: Hero, About, Skills, Projects, Services, and Contact

## Technologies Used

- HTML5
- CSS
  
- CSS3 (with CSS Variables for theming)
- JavaScript (for interactivity)

## Getting Started

1. Clone or download the repository
2. Open `endurance-portfolio.html` in your browser
3. Or run a local server: `python -m http.server 8000` and visit `http://localhost:8000`

## Customization

- Colors can be easily changed by modifying the CSS variables in `style.css`
- Content can be updated directly in `endurance-portfolio.html`
- Add more interactivity by extending `script.js`
